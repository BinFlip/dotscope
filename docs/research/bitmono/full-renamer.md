# FullRenamer

| Property | Value |
|----------|-------|
| **Protection** | `FullRenamer` |
| **Class** | `Protection` |
| **Category** | Renaming |
| **Targets** | Types, methods, fields, parameters |
| **Attributes** | `[DoNotResolve(MemberInclusionFlags.SpecialRuntime \| Model \| Reflection)]` |
| **Dependencies** | `Renamer` |

## Overview

Renames types, methods, fields, and parameters using a word-pool-based naming scheme that produces realistic-looking but garbled names. Unlike sequential renaming (e.g., Obfuscar's `A`, `B`, `C`), BitMono's renaming produces names that look like concatenated Unity/game method names.

## Algorithm

### Scope

| Member type | Condition for renaming |
|-------------|----------------------|
| Methods | Not a constructor, not virtual, not compiler-generated, not in `<Module>` |
| Parameters | All parameters of renamed methods |
| Types | Not `<Module>`, not compiler-generated, nested types only |
| Fields | Not in `<Module>`, not compiler-generated |

### Naming Format

Names are generated by `Renamer.RenameUnsafely()`, which concatenates random entries from the `RandomStrings` pool in `obfuscation.json`:

```
<word1> <word2> <word3>.<word4>
```

Example: `"HasPermission GetPlugins.Awake"`, `"Translate Start <FixedUpdate>b__4_0.get_Syntax"`

The pool contains ~300 realistic method/property names (e.g., `"HasPermission"`, `"GetPlugins"`, `"Awake"`, `"FixedUpdate"`) designed to look like legitimate Unity/game code.

### Distinctive Feature

Names contain **spaces** — unusual in .NET metadata since C# identifiers cannot contain spaces. This is the primary detection signal.

## Detection Signatures

- Symbol names containing spaces (e.g., `"HasPermission GetPlugins.Awake"`)
- Multiple symbols with names that look like concatenated C# identifier fragments
- High proportion of names matching known `RandomStrings` pool entries

## dotscope Handling

Handled by `BitMonoRenamer` technique (detection only). Detection counts names containing spaces (5+ threshold). Renaming is a lossy transformation — original names cannot be recovered.
